# image_processing_c

## About
    Image processing program in a very early stage.
    
    Currently just supports PGM and ASCII (only visualization) images.

## How to use
    ./superIMG <arg1> <arg2>(optional)
    
    To transform an image, arg1 should be an image path and arg2 should should be
        the path of the new 
    To visualize an image, arg1 should be an image path.
    To visualize a collection of images, arg1 should be a directory.

## Build
    make
